KLH Weddings & Events Website Template

Files included:
- index.html
- styles.css
- script.js
- assets/hero.jpg

How to edit:
1. Open index.html in any code editor.
2. Replace text directly in the HTML.
3. Replace assets/hero.jpg with your own image if desired.
4. Update the contact details in the Contact section.
5. If using Netlify, the enquiry form is already set up to be easily connected.

How to use in website builders:
- Framer/Webflow/Wix/Squarespace: use this as your design/content template and copy the sections into builder blocks.
- Netlify/Vercel/GitHub Pages: upload the whole folder as-is.
- WordPress: use the text and images as your page content and styling reference.

Recommended next edits:
- Add real phone number and email address
- Add Instagram link
- Replace hero image if you want a different lead image
- Add testimonials when available
- Add privacy policy/footer links before going live
